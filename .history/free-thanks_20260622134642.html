First-Time Setup

Follow these four steps the first time you use the spreadsheet. After this, the tool runs instantly each time you open it.

1

Open Apps Script

Inside the copied spreadsheet, open the Apps Script editor:

Extensions → Apps Script

A new Apps Script tab will open in your browser.

2

Run the Tool

In the Apps Script editor:

  1. Select the function: showSidebar
  2. Click the Run button (▶)

Google will request authorization the first time you run the tool.

3

Grant Permissions

When prompted, follow these steps:

  1. Click Review Permissions
  2. Select your Google account
  3. Click Advanced
  4. Click Go to GDrive To Sheet (unsafe)
  5. Click Allow
This authorization is required because the tool needs permission to read your Google Drive folder structure and write results into the spreadsheet.
🔒 No data is sent to external servers.
4

Start Using GDrive To Sheet

Return to the spreadsheet. A sidebar will appear on the right side of the sheet. You can now:

  • Select a Google Drive folder
  • Export folder and file structure
  • Generate folder paths automatically
  • Test the free version with up to 50 exported rows
